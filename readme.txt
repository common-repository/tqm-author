== Description ==

Display a feed of an authors content from TorqueMag


== Installation ==

1. Upload the `tqm-author` folder to the `/wp-content/plugins/` directory or install from the dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Drag the widget to a sidebar area and enter the author name from TorqueMag
